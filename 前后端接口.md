### Register注册（普通用户）(b:√)

- 类型：post

- 接口URL: http://localhost:8000/api/register

- f to b(前端给后端)

  ```
  {
      "name":str,
      "password":str,
      "phone":str,
      "wechat":str
  }
  ```

  例子：

  ```
  {
      "name":"test_reg",
      "password":"test",
      "wechat":"test_we"
  }
  ```

- b to f(后端给前端)

  ```
  {
  'value':int
  }
  //0成功 1s
  ```
