# 前后端接口

## 登录注册

### Register

- 类型：post

- f to b(前端给后端)

  ```
  {
      "name":str,
      "password":str,
      "phone":str,
      "wechat":str
  }
  ```

- b to f(后端给前端)

  ```
  int //0成功 1不满足数据约束条件 2 此微信号已被注册 3 此用户名已被注册
  ```

### Login

- 类型：post

- f to b(前端给后端)

  ```
  {
      "name":str,
      "password":str
  }
  ```

- b to f(后端给前端)

  ```
  {'value': value, 'type': type}
  //value:1用户名不存在 2密码错误 0成功
  //tyep:0普通用户 1管理员 -1未成功登录
  ```
